## 📌 وصف المشروع
نظام إدارة مدرسة متكامل مكتوب بلغة C++ باستخدام **الملفات الثنائية (binary files)** لتخزين البيانات.  
يدعم النظام نوعين من المستخدمين: **مدير (Admin)** و **موظف (Employee)**، مع صلاحيات مختلفة لكل منهما.

## ✨ الميزات الرئيسية
- **إدارة المستخدمين** (إضافة موظف، حذف مستخدم، تفعيل/تعطيل، عرض جميع المستخدمين، إحصائيات)
- **إدارة الطلاب** (إضافة، حذف، تحديث، عرض طالب واحد، عرض جميع الطلاب، بحث بالرقم)
- **تقرير أكاديمي** يعرض أسماء الطلاب وصفوفهم ومعدلاتهم وعدد الطلاب لكل صف
- **التحقق من صحة المدخلات** (رقم هاتف، اسم مستخدم، معدل)
- **حفظ البيانات في ملفات ثنائية (Binary)** لضمان الخصوصية والسرعة
- **برمجة هيكلية كاملة** (لا توجد كلاسات، فقط `struct` ودوال)

## 📁 هيكل المشروع (الملفات الموجودة فعلياً)
project-c-/

├── users.h # تعريفات المستخدمين والدوال

## 🛠️ متطلبات التشغيل
- مترجم C++ يدعم C++11 أو أحدث (g++, clang++, MSVC)
- نظام تشغيل: Windows / Linux / macOS

## 🚀 طريقة التجميع والتشغيل

### تجميع يدوي (موصى به نظراً لتعدد الملفات)
افتح التيرمينال في مجلد المشروع ونفذ:

```bash
g++ -std=c++11 -c main.cpp users.cpp students.cpp files.cpp validation.cpp reports.cpp
g++ -o SchoolSystem main.o users.o students.o files.o validation.o reports.o
./SchoolSystem
باستخدام Makefile (إذا كان موجوداً)
bash
make
make run

💾 التعامل مع الملفات (File Handling)
users.txt – ملف ثنائي (binary) يحتوي على بيانات المستخدمين (username, password, fullName, phone, type, active)

students.txt – ملف ثنائي (binary) يحتوي على بيانات الطلاب (id, name, grade, average)

جميع العمليات (حفظ، تحميل، تحديث، حذف، بحث) تتم عبر الملفات مباشرة، وليس عبر الذاكرة المؤقتة فقط.
